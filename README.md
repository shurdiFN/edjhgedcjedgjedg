#Pokemon App

Описание проекта

Pokemon App — это веб-приложение для покемон-энтузиастов, которое позволяет просматривать информацию о покемонах, искать их по названиям, а также фильтровать по типам. Проект предназначен для фанатов вселенной Pokémon, разработчиков, изучающих работу с публичными API, и может использоваться как учебный пример для frontend-разработки.

#Основные функции:

· Просмотр списка покемонов с пагинацией
· Поиск покемона по имени
· Фильтрация покемонов по типам (вода, огонь, трава и др.)
· Детальная информация о каждом покемоне
· Адаптивный дизайн для различных устройств

#Установка и запуск

Предварительные требования

· Node.js версии 14 или выше
· npm или yarn

#Шаги установки

#1. Клонируйте репозиторий:

```bash
git clone https://github.com/ваш-username/pokemon-app.git
cd pokemon-app
```

1. Установите зависимости:

```bash
npm install
# или
yarn install
```

1. Запустите приложение в режиме разработки:

```bash
npm start
# или
yarn start
```

1. Откройте браузер и перейдите по адресу:

```
http://localhost:3000
```

#Сборка для production

```bash
npm run build
# или
yarn build
```

#Примеры использования

Поиск покемона

Введите название покемона в поле поиска:

```bash
Например: "Pikachu", "Charizard", "Bulbasaur"
```

#Фильтрация по типу

Выберите один или несколько типов из выпадающего списка:

```bash
Доступные типы: Water, Fire, Grass, Electric, Psychic, Dragon
```

#Пример API запроса

```javascript
// Получение информации о конкретном покемоне

```
