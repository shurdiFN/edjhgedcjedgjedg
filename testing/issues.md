# Список проблем и предложений по проекту

## Критические ошибки (P0)

### REDIS-MEM-001: Утечка памяти при массовом кешировании
**Статус:**  ИСПРАВЛЕНО (v1.1.0)  
**Приоритет:** P0 (Критический)  
**Обнаружено:** 02.04.2024  
**Исправлено:** 05.04.2024  
**Ответственный:** Сагомоньян Т.А.

**Описание:**
При добавлении более 800 ключей подряд возникает ошибка выделения памяти "OOM command not allowed when used memory > 'maxmemory'".

**Шаги воспроизведения:**
```javascript
for (let i = 0; i < 1000; i++) {
    await redis.set(`key:${i}`, 'value'.repeat(100));
}
